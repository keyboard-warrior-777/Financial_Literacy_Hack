# Financial Literacy Hack — Smart Money Planner

A polished Python CLI project for **MLH Global Hack Week: Social Good** that helps users explore basic financial planning concepts in a simple, friendly terminal app.

## What it does

This project lets a user:
- calculate **simple interest**
- calculate **compound interest**
- plan a **savings goal**
- project **monthly savings growth**
- estimate an **emergency fund**
- split income using the **50/30/20 rule**
- estimate **debt payoff time**

## Why this is better than a basic calculator

Instead of a single simple formula, this version shows:
- modular Python code
- reusable functions
- input validation
- menu-driven CLI design
- clean separation of logic and user interface
- practical financial literacy concepts

## Example Use Cases

- A student checking how long it will take to save for a laptop
- A beginner understanding compound interest
- Someone estimating an emergency fund
- A user splitting monthly income into needs, wants, and savings

## Sample Features

### Savings Goal Planner
Enter:
- target amount
- current savings
- monthly contribution
- optional interest rate

The app estimates how long it may take to reach the goal.

### Budget Planner
Uses the classic **50/30/20** split:
- 50% needs
- 30% wants
- 20% savings
